Interactive Coding¶

  • Rather than keeping code scripts code and execution separate, jupyter integrates both together using the concept of code cells
  • Cells pair "snippets" of code with the output obtained from running them
  • Can contain plots/figure inline alongside code required to generate them
In [1]:
# This is a cell
#Put some code here and get some output below!

print(x)

The cell below will add our variables x=10 and z=40 together using R, and then output the sum as the variable XZ

In [7]:
%%R -i x -i z -o XZ

XZ <- x + z

The cell below will return XZ in Python! Cool, huh?!

In [8]:
